Modulos y Programación Orientada a Objetos

Unidad 6

Titulo

En la sexta tarea, utilizamos la programación orientada a objetos, que se caracteriza por un estilo de programación que nos dio algunas pautas sobre cómo usarlo. Se basa en los conceptos de clases y objetos. Este tipo de programación se utiliza para estructurar programas de software en piezas simples y reutilizables de planos de código (clases) para crear instancias individuales de objetos. Esto puede tener algo que ver con el sistema que preparamos en esta tarea, que se basa en un programa básico que almacena datos sobre vacunas, si no nos damos cuenta de que el programa se puede reutilizar en cada programa que forma la base.

¿Qué me gustó?

Al igual que con la programacion funcional del tema anterior, en esta tarea de igual forma me gusto la forma de estructurarlo (gracias a la sintaxis de python), y con el mismo poder hacer diferentes aproximaciones a la solucion mas eficiente.

Interfaz gráfica en Python

Unidad 8

Titulo

En el octavo trabajo, usamos la nueva interfaz gráfica en Python, y para ello usamos una de las bibliotecas más importantes, llamada Tkinter, que es un enlace para la biblioteca de gráficos Tcl/Tk para el lenguaje de programación Python. Se considera el estándar para la interfaz gráfica de usuario (GUI) de Python y es la instalación predeterminada de Microsoft Windows.

¿Qué me gustó?

Me gutó experimentar con los diferentes diseños y librerias que se pueden utilzar para la creación de una interfaz gráfica en Python. Personalmente, al menos para las asignaciones que tuvimos que realizar, el crear una interfaz gráfica no lo vi tan estresante como en otros lenguajes de programación.
